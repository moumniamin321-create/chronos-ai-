# Chronos AI 🧠⏱️

مساعد شخصي ذكي (Atlas) لتنظيم يومك — نسخة شخصية بالكامل، بدون حسابات،
بدون خوادم سحابية، وبدون أي تكلفة تشغيل. كل بياناتك تبقى على جهازك فقط.

## لماذا بدون Firebase أو نشر على المتاجر؟

هذه النسخة مبنية عمدًا لتكون:
- **مجانية بالكامل** — لا اشتراكات، لا رسوم نشر (25$ Google Play / 99$ سنويًا Apple)
- **خاصة تمامًا** — كل البيانات (المهام، العادات، الأهداف) مخزّنة محليًا عبر SQLite
- **تعمل بدون إنترنت** — Atlas هنا ذكاء محلي قائم على تحليل سلوكك الفعلي (إحصائيات + قواعد)، وليس نموذج لغوي خارجي

إذا رغبت لاحقًا بربط Atlas بذكاء اصطناعي خارجي حقيقي (مثل Claude)، راجع
التعليقات داخل `lib/services/ai_service.dart` — هذا مصمم كنقطة توسّع واحدة
واضحة، لكنه **اختياري** وغير مفعّل افتراضيًا.

## هيكل المشروع

```
lib/
  main.dart              نقطة الدخول
  app.dart                تركيب MaterialApp + التوجيه الأولي
  core/                   ثيم، ثوابت، مسارات، معالجة الأخطاء، الحالة العامة
  models/                 نماذج البيانات (مستخدم، مهمة، عادة، هدف، جدول)
  ai/                     "دماغ" Atlas: ذاكرة، تحليل شخصية، تنبؤ، تخطيط، توصيات
  services/                قاعدة البيانات، الإشعارات، الصوت، التخزين
  features/                كل شاشة في مجلدها الخاص (Clean Architecture مبسّطة)
```

## متطلبات البناء (على جهاز كمبيوتر)

1. تثبيت [Flutter SDK](https://docs.flutter.dev/get-started/install) (مجاني)
2. تثبيت Android Studio (لأندرويد) أو Xcode على Mac (لآيفون) — مجانية
3. داخل مجلد المشروع:
   ```bash
   flutter pub get
   flutter run
   ```
4. لإنتاج ملف APK تثبته يدويًا على هاتف أندرويد بدون متجر:
   ```bash
   flutter build apk --release
   ```
   الملف الناتج يكون في `build/app/outputs/flutter-apk/app-release.apk`

## ملاحظات

- التطبيق مستخدم واحد فقط، بدون تسجيل دخول.
- الإشعارات والصوت (تحويل كلام لنص والعكس) تعمل بالكامل على الجهاز، بدون أي API خارجي.
- ميزة OCR لقراءة المهام من الصور لم تُضمّن في هذه النسخة المبسّطة لأنها
  عادة تحتاج إما مكتبة أثقل (تكبّر حجم التطبيق) أو خدمة سحابية مدفوعة —
  يمكن إضافتها لاحقًا عبر حزمة `google_mlkit_text_recognition` (تعمل محليًا ومجانًا) إذا احتجتها.
